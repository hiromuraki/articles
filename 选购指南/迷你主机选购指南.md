## 1 概述

Linux 迷你主机是指搭载 Linux 操作系统、体积小巧（通常小于 1L）、功耗低（10-65W）的紧凑型计算设备。它整合了 CPU、内存、存储和接口，可独立运行，无需额外外设即可满足基础计算需求。与树莓派等单板计算机（SBC）相比，迷你主机通常采用 x86 架构（部分为 ARM），性能更接近传统 PC，同时保留了低功耗和便携性优势。

**核心优势**

-   **体积小巧**：多数机型尺寸如手掌大小（例如 Intel NUC 约 112×112×37mm），适合空间受限场景（如客厅、机柜）。
-   **低功耗**：典型功耗 10-30W，全年运行成本仅为传统台式机的 1/5（以 24 小时运行计算）。
-   **Linux 友好**：硬件驱动兼容性强，主流发行版（Ubuntu、Fedora、Debian）可直接安装，部分厂商甚至预装定制 Linux 系统（如 System76 的 Pop!\_OS）。
-   **多场景适配**：可作为家庭服务器（NAS、Docker 节点）、媒体中心（Kodi）、轻量级工作站（编程、文档处理）或边缘计算网关。

## 2 选购指南

### 2.1 处理器

#### 2.1.1 架构选择

**x86 架构**

-   **兼容性**：强，支持几乎所有 Linux 发行版和 x86 软件（如 Docker、VirtualBox）。
-   **适用场景**：适合需要运行复杂应用的场景（如开发环境、数据库服务器）。
-   **代表芯片**：Intel N 系列（N100/N200）、AMD Ryzen 5000U/7000U 系列。

**ARM 架构**

-   **功耗**：更低（5-10W）。
-   **兼容性**：部分软件可能需要编译适配（如某些闭源驱动）。
-   **适用场景**：适合轻量场景（如家庭服务器、物联网网关）。
-   **代表芯片**：树莓派 RP2040、Rockchip RK3588。

#### 2.1.2 性能指标

-   **CPU 核心数**：4 核起步，8 核适合多任务处理。
-   **睿频频率**：建议 ≥2.0GHz，以保证系统流畅性。
-   **TDP（热设计功耗）**：建议 ≤15W，可实现无风扇被动散热。

### 2.2 内存

#### 2.2.1 容量

-   **轻量场景**（媒体中心、单服务节点）：4GB 起步，推荐 8GB。
-   **多任务场景**（Docker 容器、虚拟机）：建议 16GB 以上，支持双通道更佳。

#### 2.2.2 类型

-   **规格**：优先选择 DDR4 或 DDR5，频率建议 ≥2400MHz。
-   **扩展性**：部分迷你主机（如 Intel NUC）支持通过 SO-DIMM 插槽更换或升级内存，选购时需确认是否支持用户自行升级。

### 2.3 存储

#### 2.3.1 内置存储

-   **eMMC**：低成本、低性能（常见于入门机型，如树莓派），适合临时系统或轻量存储。
-   **NVMe SSD**：高性能（顺序读写 ≥1000MB/s），推荐作为系统盘。部分机型支持 M.2 插槽扩展（如 Beelink SER5）。
-   **SATA SSD/HDD**：可通过外接硬盘盒或内置 2.5 英寸插槽扩展（如部分华硕 PN 系列），适合大容量存储（如 NAS 场景）。

#### 2.3.2 扩展性

-   **存储接口**：优先选择支持 M.2 NVMe + SATA 双接口的机型。
-   **外部接口**：提供 USB 3.2 Gen2（10 Gbps）接口，以连接高速外置存储。

### 2.4 接口与网络

#### 2.4.1 网络接口

-   **有线**：1Gbps 以太网为基础，2.5Gbps 或 10Gbps 适合高带宽场景（如家庭服务器、视频流传输）。
-   **无线**：Wi-Fi 6（802.11ax）+ 蓝牙 5.0 为标配，确保无线稳定性（尤其媒体中心或边缘节点）。

#### 2.4.2 视频输出

-   **标准**：HDMI 2.1（支持 4K@60Hz）或 DisplayPort 1.4，满足多屏显示或高分辨率媒体需求。

#### 2.4.3 USB 接口

-   **数量与类型**：至少 2 个 USB 3.0/3.2 接口（用于高速数据传输）+ 1 个 USB-C（支持 PD 供电或视频输出更佳）。
-   **扩展用途**：USB 2.0 接口可用于连接外设（如传感器、打印机）。

### 2.5 电源与散热

#### 2.5.1 电源

-   **类型**：优先选择内置电源（如 Intel NUC），避免外接电源适配器占用空间。
-   **功率**：若为外置电源，需确认功率是否匹配（如 65W 适配器可满足 Ryzen 5 5500U 满负载运行）。

#### 2.5.2 散热

-   **被动散热**（无风扇）：适合静音场景（如卧室媒体中心），但需注意 CPU TDP ≤10W。
-   **主动散热**（风扇）：适合高性能 CPU（如 i5-1340P），需确认风扇噪音控制（≤30dB 为安静）。

### 2.6 Linux 兼容性

#### 2.6.1 预装 Linux

-   **首选**：厂商官方支持 Linux 的机型（如 System76 Meerkat、Purism Librem Mini）。
-   **备选**：经社区验证兼容性的型号（如 Intel NUC、华硕 PN 系列）。

#### 2.6.2 避坑点

-   **硬件限制**：部分品牌（如部分联想迷你主机）可能锁死 BIOS 或缺乏 Linux 驱动。
-   **验证建议**：购买前需查阅 Linux Hardware Database 或 Reddit r/linux 社区评价。
