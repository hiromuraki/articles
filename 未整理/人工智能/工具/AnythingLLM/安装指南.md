## 1 Linux

### 1.1 配置 Docker 环境

**安装 Docker**

> 见 Docker 相关

### 1.2 创建 Docker 文件

**Dockerfile**

```dockerfile
FROM mintplexlabs/anythingllm:latest
```

**docker-compose.yml**

```yml
name: anything-llm

services:
    main:
        build: .
        container_name: anything-llm-main
        cap_add:
            - SYS_ADMIN
        restart: unless-stopped
        ports:
            - "3001:3001/tcp"
        environment:
            - STORAGE_DIR=/app/server/storage
        volumes:
            - ${USER_LOCAL_SHARE}/anythingllm:/app/server/storage
            - ${USER_LOCAL_SHARE}/anythingllm/.env:/app/server/.env
```

### 1.3 启动服务

```bash
docker compose up --build
```
