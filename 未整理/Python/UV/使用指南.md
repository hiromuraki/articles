## UV 项目结构

```
example
├── .python-version
├── README.md
├── main.py
└── pyproject.toml
```

## 基本操作

**创建项目**

创建 example 项目目录

```bash
uv init example
```

**添加/移除包**

添加 numpy 包

```bash
uv add numpy
```

移除 numpy 包

```bash
uv remove numpy
```

**更新项目环境**

```bash
uv sync
```

**更新 lockfile**

```bash
uv lock
```

**运行**

```bash
uv run main.py
```

### 常用环境变量

环境变量参考：[Environment variables | uv](https://docs.astral.sh/uv/reference/environment/)

UV_NO_CACHE

禁用缓存

| 值   | 说明  |     |
| --- | --- | --- |
| 0   | 禁用  |     |
| 1   | 启用  |     |

UV_LINK_MODE

设置链接模式

| 值    | 说明        |
| ---- | --------- |
| copy | 复制模式，禁用链接 |

UV_COMPILE_BYTECODE

编译字节码

| 值   | 说明  |
| --- | --- |
| 0   | 禁用  |
| 1   | 启用  |
