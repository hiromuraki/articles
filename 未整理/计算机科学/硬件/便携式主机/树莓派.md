## 1 概述

树莓派（Raspberry Pi）是一款由英国树莓派基金会（Raspberry Pi Foundation）开发的微型单板计算机，旨在以低成本、高灵活性的硬件推动计算机科学教育。自 2012 年发布以来，它已成为全球创客、开发者和教育者的热门工具，被广泛应用于编程学习、物联网、自动化、媒体中心等领域。

### 1.1 特点

-   **低成本**: 价格通常在几十到几百元人民币（如 Raspberry Pi Zero 约 5 美元，Raspberry Pi 5 约 75 美元）
-   **小巧便携**: 尺寸接近信用卡大小（如 Raspberry Pi 4 尺寸为 85mm x 56mm）
-   **开源生态**：支持 Linux 系统（如 Raspberry Pi OS、Ubuntu）和丰富的开源软件
-   **强大的扩展性**:提供 GPIO（通用输入输出）引脚，可连接传感器、电机、显示屏等外设

### 1.2 常见用途

-   **教育与编程**
    -   学习编程：学习 Python、C/C++、Scratch 等编程语言。
    -   数据分析：运行 Jupyter Notebook 进行数据分析。
-   **物联网（IoT）**
    -   环境监测：连接温湿度传感器（DHT22）、摄像头（CSI 接口）构建环境监测系统。
    -   云端通信：通过 MQTT 协议与云端通信（如阿里云、AWS IoT）。
-   **家庭娱乐**
    -   媒体中心：安装 Kodi 或 Plex 打造 4K 媒体中心。
    -   复古游戏机：搭建 RetroPie 复古游戏机（支持 PS1 以下主机模拟）。
-   **服务器应用**
    -   个人服务：部署个人博客（WordPress）、私有云（Nextcloud）。
    -   网络工具：运行 Pi-hole 屏蔽广告或搭建 VPN（WireGuard）。
-   **硬件项目**
    -   机器人控制：驱动机械臂、智能小车（通过 GPIO 控制电机和舵机）。
    -   创意装置：制作智能魔镜（MagicMirror² 框架）或 LED 艺术装置。

## 3 操作系统推荐

-   **Raspberry Pi OS**：官方系统，兼容性最佳，适合新手。
-   **Ubuntu Core**：轻量级 Linux，适合服务器和容器化应用。
-   **LibreELEC/OSMC**：专为媒体中心优化的 Kodi 系统。
-   **BalenaOS**：支持物联网设备集群管理。
-   **Windows IoT Core**：微软官方轻量级 Windows 系统（仅限特定型号）。
