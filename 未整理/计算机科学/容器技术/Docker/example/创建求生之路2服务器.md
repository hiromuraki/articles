## 创建文件

```bash
mkdir l4d2-server && cd l4d2-server
```

目录文件结构如下

```
l4d2-server
├── Dockerfile
└── docker-compose.yml
```

**Dockerfile**

```dockerfile
FROM cm2network/steamcmd

VOLUME ["/app/game"]
```

**docker-compose.yml**

```yml
name: l4d2-server

services:
    dedicated-server:
        build: .
        container_name: l4d2-dedicated-server
        ports:
            - "27015:27015/udp"
            - "27015:27015/tcp"
        volumes:
            - type: bind
              source: /home/user/game_server/Left_4_Dead_2
              target: /app/game
              read_only: false
        restart: unless-stopped
        command: |
            /app/game/srcds_run \
              -game left4dead2 \
              -insecure \
              -port 27015 \
              -tickrate 64 \
              -debug \
              +exec server.cfg \
              +nomaster \
              +map c1m2_streets
```

volumes 字段中需要将本地的求生之路 2 服务端文件目录挂载到容器的 /app/game

## 启动服务

```bash
docker compose up --build
```
