```bash

```

## 3. 镜像管理

### 3.1. 导出镜像

```bash
podman save --output nginx.tar nginx:latest
```

### 3.2. 导入镜像

```bash
podman load --input nginx.tar
```

```bash
cat nginx.tar | podman load
```
