# Steamcmd 安装指南

## 1. Linux

### 1.1. Debian 系

```bash
sudo add-apt-repository multiverse
sudo dpkg --add-architecture i386
sudo apt update
sudo apt install steamcmd
```

## 2. Windows

二进制包下载地址：<https://steamcdn-a.akamaihd.net/client/installer/steamcmd.zip>

创建 steamcmd 目录，解压文件即可通过 `steamcmd.exe` 使用
