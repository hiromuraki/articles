# 双筒望远镜选购指南

## 1. 基本参数

### 1.1. 倍率（Magnification）

含义：将物体拉近的程度。

例如 10x，意味着 1000 米外的物体看起来像在 100 米处。

需要注意的是，尽管高倍率意味着看得更远，但倍率越高也意味着越难控制，高倍率下手部微小的晃动都会导致画面大幅晃动。

选择建议：如果用于手持，最好选择 6x - 12x 之间的倍率（通常 10x 是手持的最高倍率建议），更高的倍率需要使用三脚架等固定设备配合使用。

### 1.2. 物镜口径（Objective Lens Diameter）

含义：前端镜片的直径（mm）。

口径越大，进光越多，分辨率（理论极限）越高，能看到的效果往往更好。当然越大的口径也意味着越大的体积和重量，如果需要长时间手持且没有对应需求，无需刻意追求大物镜口径。

### 1.3. 棱镜系统 (Prism System)

作用：让望远镜能看到正立的图像。主要有两种系统：保罗式与屋脊式。

![棱镜系统对比](./assets/棱镜系统对比.jpg)

- **保罗式 (Porro)**： 最传统的“W”形状。
![保罗式例图](./assets/保罗式例图.jpg)
  优点： 结构简单，便宜就能做到很好的透光率和立体感。
  缺点： 傻大黑粗，重，携带不便。Z 型号光路可能会影响对距离的判断。

- **屋脊式 (Roof)**： 直筒形状，现代主流。
![屋脊式例图](./assets/屋脊式例图.jpg)
  优点： 便携，好看。光线的出入都在同一条直线上，在距离感、体视感上更接近人的肉眼。
  缺点： 制造难度大，为了弥补光路缺陷需要付出额外成本才能实现和保罗式相同的效果，因此同价位下光学素质通常略输保罗式。

### 1.4. 棱镜材质 (Glass Type)

作用：好的棱镜材质可以增加折射率，降低光线损失，提高成像效果。目前主要有 BaK4 与 BK7 两种材质。

| 材质类型 | 说明                                               |
| -------- | -------------------------------------------------- |
| BK7      | 折射率较高，但出瞳边缘有切边，光损失多。优点是便宜 |
| K9       | BK7 在国内的代名，可视为等同于 BK7                 |
| BaK4     | 折射率很高，光损失少，出瞳无切边。但价格稍贵       |

![棱镜材质对比](./assets/棱镜材质对比.jpg)

### 1.5. 增透膜（Coating）

作用：增透膜主要用于降低光线的反射，增强呈现亮度与对比度。

当光线从空气进入玻璃/玻璃进入空气时会有一部分被反射掉，如果望远镜没有增透膜就会丢失相当一部分的入射光线，导致成像偏暗，对比度偏低。为了降低反射，需要在望远镜中配置单层甚至多层增透膜，通常多层膜效果好于单层膜。

一般单层膜呈红色或蓝紫色，多层膜呈淡淡的绿色或暗紫色。但需要注意有部分单层膜也会呈绿色（仅装饰用）。

| 镀膜类型                            | 说明                                                           |
| ----------------------------------- | -------------------------------------------------------------- |
| 镀膜（Coated Optics）                | 表示在一个表面上有单层增透膜（通常是物镜和目镜外表面）         |
| 全表面镀膜（Fully Coated）           | 所有镜片和棱镜表面都镀了单层膜                                 |
| 多层镀膜（Multi-Coated）            | 至少在一个表面上镀有多层膜，其他表面可能只有单层膜甚至没有镀膜 |
| 多层全表面镀膜（Fully Multi-Coated） | 所有表面都有增透膜，且部分（甚至全部）表面使用了多层膜         |

![镀膜对比](./assets/镀膜对比.jpg)

选择镀膜时认准 FMC (Fully Multi-Coated，全表面多层镀膜)。

### 1.6. 出瞳直径（Exit Pupil）

含义：目镜对着眼睛射出的那束光柱的粗细。

> 人眼在暗环境下瞳孔会扩大到 5-7mm。

估算公式：物镜直径 ÷ 倍数。例如对于 10 x 25 的镜子，出瞳直径可能只有 25 / 10 = 2.5mm。就像用吸管看世界。出瞳直径最好在 4mm - 5mm 以上。比如 10 x 50 或者 8 x 42。

> 物镜直径 / 出瞳直径 = 倍数，因此可以通过该公式来反向推测倍数是否有虚标。

### 1.7. 出瞳距离（Eye Relief）

含义： 眼睛必须离目镜多远才能看清整个视场。

此参数主要影响戴眼镜群体的感受。戴眼镜者需要长出瞳距离（> 15mm）。因为眼镜片挡着，眼睛贴不上目镜。如果出瞳距离太短，戴着眼镜看时视野周围会有一圈黑边，只能看到中间一点点。

### 1.8. 视场（Field of View, FOV）

含义：你能看到的画面有多宽。

通常有两种表示方式，一种是角度（如 6.5∘），一种是千米处视野宽度（如 114m/1000m）。可通过以下公式粗略换算：

$$ 角度 \times 17.5 \approx 1000米处的视野宽度（米） $$

> 通常视场和倍率是反比关系，越高的倍率往往意味着越低的视场。

### 1.9. ED 镜片（Extra-low Dispersion）

作用：降低色散。

看远处的路灯或月亮边缘时，看到一圈紫边或绿边就是色散，通常亮度对比越高色散越严重。ED 镜片可以降低色散，保持颜色纯净。

### 1.10. 调焦（Focus）

保罗式望远镜通常由两种调焦模式：中央调焦（中调）与双镜独立调焦（双调）。

中央调焦即通过设备中央的调焦轮对双镜统一调焦，双调即独立对双镜进行调焦。

虽然因为结构问题，双调望远镜通常有更好的防护能力，但双调使用起来也比较繁琐。如果没有明确需求，通常选择中调望远镜即可。

## 2. 特殊说明

### 2.1. 倍率与口径选择

- 8×20、8×25、10×25：适合出门长途旅行或者野外求生以及看演唱会
- 8×30、8×32：适合短途旅游、运动爬山
- 8×40、8×42：规格光学最好、适合日常使用
- 7×50：天文学爱好者使用，适合看星空，海航使用
- 10×42、10×50：高倍数、固定地方使用

### 2.2. 棱镜系统选择

保罗式结构简单，采用 Z 型折叠光路，容易实现光线的高反射，缺点是体积更大更笨重；而屋脊式虽然体积更紧凑，但其光路设计存在天然弱势，需要花费更多成本去抵消其光路缺陷。

因此，如果预算有效（500以下），优先考虑购买保罗式的结构，同样的价格下相比屋脊式能买到光学性能更好的产品。

需求适配：

- 保罗式：观星
- 屋脊式：旅行

## 3. 注意事项

### 3.1. 严禁使用望远镜观看太阳

由于望远镜有聚集光线的能力，物镜朝向太阳时会导致目镜温度急剧升高，此时若使用望远镜会在极短时间内对眼睛造成严重物理损伤。故任何时候都**不得使用望远镜观看太阳**。
